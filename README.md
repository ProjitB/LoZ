Legend of Zelda 3D Game
=========================

Controls:

   Arrow Keys(Moves relative to the ships current position):
      
      Up :: Move Forward
      Down :: Move Backward
      Right :: Move Right
      Left :: Move Left

   Mouse Scroll:

      Scroll Up :: Zoom In
      Scroll Down :: Zoom Out

   Numbers:

      1 :: Follow - Cam View
      2 :: Top View
      3 :: Tower View
      4 :: Boat View
